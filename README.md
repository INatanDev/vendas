## Visão Geral  
O projeto **vendas** é um sistema para controle de vendas de produtos, criado para fins de aprendizado e utilidade prática. Ele oferece funcionalidades para gerenciar produtos, pedidos, clientes e vendas, com backend e frontend integrados.

---

## Tecnologias Utilizadas  
- Back-end: Java  
- Front-end: TypeScript ( React )  
- Estilo/CSS: CSS   
- Banco de dados, frameworks etc. — postgresql, spring boot, next.js
- Gerenciamento de versões: Git  
- Licença: MIT 

---

## Arquitetura do Projeto  
O repositório está dividido em dois sub-projetos principais:  
- `vendas-api` — O módulo de API / backend  
- `vendas-app` — A aplicação cliente (frontend)  

Cada um com sua própria configuração, dependências e execução.
